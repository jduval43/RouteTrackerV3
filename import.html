Route Info
Optionally update the route name and description shown in the header.
↑ Upload CSV File
Upload a .csv file with your stop data. First row can be a header and will be skipped automatically. Columns are detected by name or position.
Accepted column names (case-insensitive):
Stop # or seq: seq stop # stop_num
Address: address addr street primary
Street name: street_name complete full_address
Notes: note notes comment

If no header row: assumes col 1 = seq, col 2 = address, col 3 = note
USPS LFA220P2 format: seq in col 1, primary # in col 2, street name in col 3 — auto-combined.
✎ Manual Text Entry
Type or paste your stops. One stop per line. Flexible formats accepted:
123 Main St — address only
5, 123 Main St — stop number, address
5, 123 Main St, dog at door — stop, address, note
123 Main St | leave at door — pipe separator for note
5 | 123 Main St | packages go to porch — pipe with seq
Blank lines and lines starting with # are ignored.
